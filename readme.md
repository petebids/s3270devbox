# basic dev environment 

## Setup

```
git clone https://github.com/petebids/s3270devbox.git

cd s3270devbox

vagrant up

```



# Accessing the box 
simple ssh access via vagrant

```
vagrant ssh
```

raw ssh usage for vscode ssh plugin - default creds are vagrant/vagrant
```
ssh vagrant@127.0.0.1 -p 2222
```
